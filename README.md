# Input Reduction for Scalable Reinforcement Learning in Quantum Computing Noise Models

In my bachelor thesis I apply Proximal Policy Optimization (PPO) to accelerate quantum noise modeling through an input reduction strategy. By optimizing the learning pipeline, the approach achieved a 50% reduction in execution time for 1-qubit systems and 43% for 3-qubit systems compared to the baseline method.

A further analysis was carried out on the shot dependence of model performance to identify practical measurement thresholds for real-device implementations. The results showed that input reduction can be a promising and scalable strategy for deploying reinforcement learning methods on quantum hardware.
